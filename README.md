# 🎤 Interview AI — AI Powered Mock Interview System

An intelligent AI-powered interview platform that generates role-specific interview questions, evaluates user answers using LLMs, and provides detailed performance feedback with scoring, strengths, weaknesses, and improvement suggestions.

---

# 📌 Problem Statement

Many candidates struggle with interview preparation due to lack of personalized feedback and realistic interview simulation.

Interview AI solves this by using LLM-powered question generation and answer evaluation to simulate real technical interviews and provide AI-driven performance analysis.

---

# ✨ Features

✅ Resume PDF Upload
✅ Job Description-Based Question Generation
✅ AI-Powered Technical & Behavioral Questions
✅ Interactive Interview Flow
✅ LLM-Based Answer Evaluation
✅ Communication & Technical Scoring
✅ Skill Detection from Answers
✅ Detailed Performance Report
✅ AI-Based Improvement Suggestions
✅ Downloadable Interview Report

---

# 🧠 System Workflow

1️⃣ Upload Resume PDF
2️⃣ Paste Job Description
3️⃣ AI Generates Personalized Interview Questions
4️⃣ User Answers Questions
5️⃣ LLM Evaluates Responses
6️⃣ Performance Report is Generated

---

# 🏗️ Architecture

Resume Parser → Question Generator → Interview Engine → Answer Evaluator → Report Generator

---

# 🛠️ Tech Stack

| Category          | Technology         |
| ----------------- | ------------------ |
| Frontend          | Streamlit          |
| Backend           | Python             |
| LLM               | Groq API (LLaMA 3) |
| AI Framework      | LangChain          |
| PDF Processing    | PyPDF              |
| Report Generation | FPDF               |
| AI Logic          | Prompt Engineering |

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/Palakbiradar119/interview-ai.git

cd interview-ai
```

## Create Virtual Environment

```bash
python -m venv venv

venv\Scripts\activate
```

## Install Requirements

```bash
pip install -r requirements.txt
```

---

# 🔑 Configure API Key

Create `.env` file:

```env
GROQ_API_KEY=your_api_key
```

---

# ▶️ Run Application

```bash
streamlit run app.py
```
